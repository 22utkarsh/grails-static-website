Sponsors

The Grails project is generously sponsored by several companies:

  • OCI employs key members of the Grails team
  • Pivotal provide website hosting
  • JFrog provide the infrastructure for deploying and hosting our snapshots and releases, thanks to the Bintray social platform for distribution, and the OSS Artifactory repository.
  • YourKit supports open source projects with its full-featured Java Profiler. YourKit, LLC is the creator of YourKit Java Profiler and YourKit .NET Profiler, innovative and intelligent tools for profiling Java and .NET applications.
